Emulate spatially resolved internal temperature variability trained on climate model large ensemble data. Contains scripts for data pre-processing, running the emulator, and generating diagnostic figures. Contact Marissa Saenger (msaenger@ucsd.edu) with questions and/or feedback.

- InVERT_emulator.ipynb: notebook for running the emulator.
- InVERT_Diagnostics_1: notebook for producing figures 2-4; figures S3-S5, S7, S8, S11, S12, and S19. 
- InVERT_Diagnostics_2: notebook for producing figures S17 and S18.
- InVERT_Diagnostics_3: notebook for producing figures S1, S2, S6, and S9.
- InVERT_Diagnostics_4: notebook for producing figures S13 - S16.
- InVERT_Diagnostics_5: notebook for producing figure S20.
- InVERT_Diagnostics_6: notebook for producing figure S10.
